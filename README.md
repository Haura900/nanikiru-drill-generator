# 類題何切る量産ドリル

何切る問題を出題・復習しながら、元問題から類題を自動生成するブラウザアプリです。

## できること

- ジャンル別に問題を出題
- 忘却曲線に基づいた復習出題
- 元問題を反転・置換・スライドして類題を自動生成
- 何切るシミュレーターで元解答と類題を検証
- 成績をジャンル別・日付別に確認
- 問題、回答状況、成績をブラウザの `localStorage` に保存
- 保存内容を Base64 で出力・復元

## 保存について

問題や学習履歴は、このブラウザの `localStorage` にだけ保存されます。サーバー側の保存やパスワードは使いません。

## 使い方

1. `launch_admin.bat` を実行して起動します。
2. 「問題作成」で手牌、副露、ドラ表示牌、指定解答、巡目などを入力して検証します。
3. 条件を満たしたら元問題を登録するか、類題を生成・登録します。
4. 「出題」でジャンル別出題や復習出題を行います。
5. 「データ出力」から Base64 を保存してバックアップできます。

## 問題作成のルール

- 同じ手牌と副露の組み合わせは重複登録できません。
- 指定解答は複数個登録できます。
- 類題生成では、元問題を変換しても指定解答の乖離率が許容範囲内に収まる候補だけを採用します。

## 使っている素材・仕組み

- 何切るシミュレーター: [mahjong-cpp](https://github.com/nekobean/mahjong-cpp)
- 牌画像: [麻雀牌画像素材](https://majandofu.com/mahjong-images)

## テスト

```powershell
python -m unittest discover -s tests -v
```

## WebAssembly の再ビルド

シミュレーター本体を更新したら、`docs/wasm/` を再生成します。

```powershell
powershell -ExecutionPolicy Bypass -File scripts/build_wasm.ps1
```

## 補足

GitHub Pages ではアプリ本体を公開できますが、問題データはブラウザ内にだけ保存されます。公開用の暗号化ファイルやパスワード入力は使いません。
